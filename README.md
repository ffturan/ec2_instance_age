# ec2_instance_age
Lists top 10 newest and oldest EC2 Instances by launch time.  
Uses PrettyTable.
Requires AWS profile and region as input.
## Usage
./ec2_instance_age profile-name region-name
## Output
```shell
-----------------------

-----------------------

+---------------------+----------------------------+------------+----------------+----------------+
-----------------------
Oldest 10 EC2 Instances
-----------------------
+---------------------+----------------+-----------+--------------+----------------------------+
```
